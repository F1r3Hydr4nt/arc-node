# arc-node

Planning and integration repo for **signed, secure** BSV transaction broadcasting using [ARC](https://github.com/bitcoin-sv/arc) (bitcoin-sv/arc) and [Teranode](https://github.com/bsv-blockchain/teranode) (bsv-blockchain/teranode).

- **arc/** – [ARC](https://github.com/bitcoin-sv/arc) (Authoritative Response Component) source; broadcast & status API, Metamorph, BlockTx.
- **[BROADCAST_RESEARCH_AND_PLAN.md](./BROADCAST_RESEARCH_AND_PLAN.md)** – Research on broadcast capabilities of both repos and a phased plan to implement signed, secure broadcast (ARC-centric; Teranode as node backend).
